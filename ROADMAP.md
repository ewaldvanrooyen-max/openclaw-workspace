## 🗓️ The Road Ahead

### Phase 1: Foundation ✅
**Status:** Complete — Live!

- [x] Landing page live at [pocketpal.co.za](https://interlinkedcapital.co.za/pocketpal/)
- [x] 8-step onboarding flow (language → personal info → needs → people → keywords → memory)
- [x] Core Nanobot framework (Router + base architecture)
- [x] SQLite + sqlite-vec for memory storage
- [x] Privacy-first design (local only)

---

### Phase 2: Shadow Memory 🔄
**Target:** Q2 2026

**What it means:**
PocketPal learns from your patterns — not by spying, but by remembering what *you* tell it matters. It notices when you usually wake up, who you call most, and what topics you care about.

- [ ] Pattern recognition engine
- [ ] Weekly check-ins for important people
- [ ] "Remember when..." functionality
- [ ] Context-aware suggestions

---

### Phase 3: Ghostwriter ✍️
**Target:** Q3 2026

**What it means:**
PocketPal learns YOUR writing style. Emails, WhatsApp messages, Slack replies — it writes like *you*, not like a robot.

- [ ] Tonal mimicry AI
- [ ] Style profile builder
- [ ] Context-aware tone adjustment
- [ ] Privacy-preserving training (local only)

---

### Phase 4: Financial Guardian 💰
**Target:** Q3-Q4 2026

**What it means:**
An AI that actually watches your back. Guardian monitors spending patterns and steps in when things look off — suspicious transactions, unusual subscriptions, budget overruns.

- [ ] Transaction pattern analysis
- [ ] Smart alerts (not spam)
- [ ] Budget tracking
- [ ] Intervention prompts

---

### Phase 5: Adaptive UX 🎨
**Target:** Q4 2026

**What it means:**
PocketPal gets smarter about how it presents information. Dark mode, larger text, simplified views — whatever you need.

- [ ] Dynamic UI scaling
- [ ] Accessibility features
- [ ] Personalized dashboard
- [ ] Notification preferences

---

### Phase 6: Vision 👁️
**Target:** 2027

**What it means:**
Computer vision capabilities — read receipts, scan documents, describe images. Your AI can now *see*.

- [ ] OCR for documents
- [ ] Image description
- [ ] Visual context memory

---

### Phase 7: Privacy Bridge 🌉
**Target:** 2027

**What it means:**
Connect securely with other PocketPals without giving up your data. Federated learning meets social networking.

- [ ] A2A (Agent-to-Agent) protocol
- [ ] Secure data sharing (zero-knowledge)
- [ ] Selective memory sync
- [ ] Decentralized identity

---

### Phase 8: Voice & Creative 🎤
**Target:** 2027

**What it means:**
Your AI talks like you. Voice cloning, TTS, creative writing — PocketPal becomes truly expressive.

- [ ] Voice cloning (local)
- [ ] Natural TTS
- [ ] Creative writing assistant
- [ ] Story generation

---

### Phase 9: Social Mesh 🌐
**Target:** 2027-2028

**What it means:**
Connect with friends. PocketPal Circle, Ghost Group Chats, and federated social features — AI that brings people together.

- [ ] PocketPal Circle (friend groups)
- [ ] Ghost Group Chats (AI-mediated)
- [ ] Community features
- [ ] Federated social network

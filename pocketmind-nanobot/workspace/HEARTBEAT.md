# Periodic Tasks

PocketMind can perform periodic tasks to stay updated on your needs.

## Quick Tasks (respond directly)

- [ ] Check if user profile needs updating
- [ ] Confirm I'm remembering you correctly

## Long Tasks (use spawn for async)

- [ ] Review conversation history and offer to save important points
- [ ] Check if there are any profile updates to suggest

---

*Heartbeat runs every 30 minutes when gateway is active.*

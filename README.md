## ✨ What is PocketPal?

PocketPal is your **personal AI companion** — built for real people in the real world. Unlike corporate AI assistants that treat you like a product, PocketPal actually gives a damn about *you*.

It's witty, encouraging, and remembers the things that matter. Built with ❤️ by the [WolfPack Empire](https://github.com/ewaldvanrooyen-max) and designed specifically for South Africans (and anyone who wants an AI that feels human).

## 🚀 Getting Started

### Prerequisites

- **Android device** with Termux installed, OR
- **Linux/WSL** desktop environment
- ~50MB storage (including AI models)

### Quick Install

```bash
# Clone the repo
git clone https://github.com/ewaldvanrooyen-max/pocketmind-agentmesh.git
cd pocketmind-agentmesh

# Run the setup
./setup.sh

# Launch PocketPal
./run.sh
```

### First Launch

1. Select your language (English, Afrikaans, Zulu, Xhosa, or Sotho)
2. Tell PocketPal about yourself (name, job, location)
3. Add people who matter to you
4. Set keywords for reminders
5. **You're ready!** 🎉

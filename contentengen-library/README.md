# ContentEngen Local Library

## Structure
```
contentengen-library/
├── videos/          # Video assets
├── images/         # Image assets  
└── audits/          # Quality audit reports
```

## Naming Convention
Format: `[Volume]_[Aesthetic]_[Subject]_[Type].mp4|.jpg`

### Volumes
- Sovereign (Professional/Corporate)
- Solarpunk (Eco-futuristic)
- Sanctuary (Calm/Wellness)
- Glitch (Dark/Urban/Tech)

### Example
- `Sovereign_Corporate_Meeting_Wide.mp4`
- `Glitch_Neon_City_Rain.mp4`

## Audit Reports
Each video/image has a corresponding `.md` audit file in `audits/`
- Quality rating
- Content description
- Issues/notes
- Library tags
